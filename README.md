# ApolloGraphQL-Express-MongoDB-Template

Quick start your backend with this template:

- Apollo GraphQL
- Express
- MongoDB
- bcrypt
- jwt
- Working authentication process (register, login...)

Simply fill in your mongodb detail in .env file and you are good to go
`DB_USER = 
DB_PASS = 
DB_ADDRESS = 
DB_NAME_DEV = 
JWT_SECRET = `
